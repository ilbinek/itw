Current work

Currently most of my time goes into JRD - Java runtime decompiler, which is a project I am working on during my internship.

My main work is fixing current bugs and adding new features, that make it easier to reverseengineer java programs while running in JVM, or just as plain JARs.

The upcoming feature is support for nested JARs. Listing and reading of all classes is implemented, you can monitor the progress on the Pull Request.

Support for rewriting of nested JARs will be added in the near future!
